# yiwn-shortid  

> Amazingly short non-sequential human-friendly unique id generator.

## Install

```sh
npm install yiwn-shortid
```

## Usage

```js
const shortid = require('yiwn-shortid');

console.log(shortid());
//PPBQWA9
```

## API

#### `shortid :: * -> String`
